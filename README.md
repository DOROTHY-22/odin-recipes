# Odin Recipes

A simple recipe website built as part of The Odin Project Foundations curriculum.

## Project Description

This project is a basic HTML website that contains a main index page and three individual recipe pages. The goal is to practice building web pages using HTML boilerplate, links, images, headings, lists, and folder structure.

## Recipes Included

- Creamy Chicken Lasagna
- Egg Roll Bowl
- Tortilla Wrap Taco Pizza

## Project Structure

```bash
odin-recipes/
├── index.html
└── recipes/
    ├── creamy-chicken-lasagna.html
    ├── egg-roll-bowl.html
    └── tortilla-wrap-taco-pizza.html
```

## What Each Page Includes

### `index.html`
- A standard HTML boilerplate.
- An `h1` heading with the text `Odin Recipes`.
- Links to each recipe page.

### Each recipe page
- A standard HTML boilerplate.
- An `h1` heading with the recipe name.
- A free image of the finished dish.
- A `Description` section with one or two paragraphs.
- An `Ingredients` section with an unordered list.
- A `Steps` section with an ordered list.
- A link back to the home page.

## Learning Goals

By completing this project, I practiced:
- Creating HTML boilerplate.
- Linking pages together.
- Organizing files into folders.
- Using headings, paragraphs, lists, and images in HTML.

## Note

This project focuses on HTML structure only. Styling can be added later with CSS.
